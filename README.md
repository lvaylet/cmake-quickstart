# CMake & GoogleTest Quickstart

## Usage

Install `clang` and `cmake`, then:

```
$ cmake -S . -B build
$ cmake --build build
$ (cd build && ctest)
```

From Visual Studio Code: `Ctrl+Shift+P` > `CMake: [Configure|Build|Debug|Run Without Debugging|Run Tests]`
